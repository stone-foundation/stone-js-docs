# Function: addMetadata()

```ts
function addMetadata(
   context, 
   key, 
   value): void;
```

Add metadata on a given decorator context.

## Parameters

### context

`DecoratorContext`

The decorator context where metadata is being added.

### key

`PropertyKey`

The key for the metadata entry.

### value

`unknown`

The value of the metadata entry.

## Returns

`void`
