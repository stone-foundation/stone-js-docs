# Function: setMetadata()

```ts
function setMetadata(
   context, 
   key, 
   value): void;
```

Set metadata on a given decorator context.

## Parameters

### context

`DecoratorContext`

The decorator context where metadata is being set.

### key

`PropertyKey`

The key for the metadata entry.

### value

`unknown`

The value of the metadata entry.

## Returns

`void`
